# 🧬 Breast Cancer Survival Analysis
EDA, Visualization &amp; ML on Breast Cancer Dataset

## 📌 Overview
This project analyzes a breast cancer dataset using:
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Models

## 📊 Dataset
The dataset contains patient details such as:
- Age
- Cancer Stage
- Tumor Size (t)
- Nodes (n)
- Metastasis (m)
- Survival Time

## 🧹 Data Preprocessing
- Handled missing values using median
- Cleaned dataset for analysis

## 📈 EDA & Visualization
Performed:
- Histogram (Age Distribution)
- Bar Graph (Cancer Stage)
- Scatter Plot (Age vs Survival Time)
- Heatmap (Correlation)
- Box Plot & Violin Plot

## 🤖 Machine Learning Models
Used:
- Linear Regression (R² = 0.81)
- Decision Tree Regressor
- Random Forest Regressor

## 🔍 Key Insights
- Higher cancer stage → lower survival time
- Tumor features strongly affect survival
- Strong correlation between t, n, m

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📄 Research Paper
Included IEEE-format research paper in the repository.

## 🚀 Conclusion
This project demonstrates how data science can be used in healthcare to analyze and predict survival outcomes.
